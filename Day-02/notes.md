## Data Analysis vs Data Science

### Data Analysis
- Studying data to find useful information and answers
- Helps in decision making
- Involves collecting, cleaning, analyzing data, and creating reports

Tools used:
- Excel
- SQL
- Python
- Power BI / Tableau

Example:
- Finding which product sells the most
- Finding which day sales are highest

### Data Science
- Uses data along with programming and mathematics
- Focuses on prediction and future outcomes
- Involves machine learning and model building

Example:
- Movie recommendations
- Predicting exam scores
